Proyectos

Whac-a-mole

Un simple pero divertido juego. Este fué mi proyecto final para el bootcamp de Frontend Masters. El mismo fué creado con Scss y Javascript muy interesante, creando objetos y calculando el tiempo que tardaba cada topo en aparecer y desaparecer.

Calculadora

Una calculadora buena, bonita y barata. Se le aplicó un poco de 'glassmporphism' porque me pareció interesante en el momento.

Visor de perros

¿Fanático de los perros? Esta página utiliza una base de datos externa para mostrarnos las distintas razas conocidas. Además agregué un poco de interactividad con el mouse en la imagen, para darle un poco más de dinamismo.

Pelotitas

Este proyecto demuestra como se puede jugar con la interactividad en una página web, las pelotas recorren libremente en la pantalla e interactúan con el mouse, siempre que este se encuentre cerca.
